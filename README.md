# Official Website of the HomeworkSMP

## Overview
HomeworkSMP is a Minecraft SMP server dedicated to providing a fun and engaging multiplayer experience. This website serves as a hub for information, updates, and community engagement related to the server.

## Development
To help contribute to this repo, you can always make a fork of this repo (or a new branch for HomeworkSMP members). When you open a PR, our service provider, Cloudflare Pages, will automatically create a preview.

If you want to build locally, pull the repository on to your computer. You will need to have [Node.js](https://node.js.org) installed. Once you have installed Node.js and pulled the repository, simple run `npm install` or `npm i`. To start a local server, run `npm run dev`.
